# Netty

Netty是什么？

* 异步事件驱动框架，用于快速开发高性能服务端和客户端。
* 封装了JDK底层BIO，NIO模型，提供了高可用的API。
* 自带编解码器解决拆包粘包问题，用户只用关心数据逻辑。
* 精心设计的reactor线程模型支持高并发海量连接。
* 自带各种协议栈。
